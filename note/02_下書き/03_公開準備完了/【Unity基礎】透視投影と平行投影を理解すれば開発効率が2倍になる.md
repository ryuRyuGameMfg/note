## 1. そもそも「投影」って何？

3Dの世界にあるオブジェクトを、2Dの画面（モニター）に映し出すこと、これが**「投影」**なんです。

この「投影」の方法には、主に2つの方式があります。

**Unity公式ドキュメント - カメラの概要：**
https://docs.unity3d.com/ja/2019.4/Manual/CamerasOverview.html

## 2. 透視投影（Perspective）とは

**読み方：** とうしとうえい

**英語：** Perspective Projection

### 特徴

透視投影は、**人間の目で見たときと同じような見え方**をする投影方式です。

・遠くのものが小さく見える
・近くのものが大きく見える
・視点からの視線が一点に収束する
・リアルな立体感が出る

道路を真っすぐ見ると、遠くの方が狭く見えますよね。あれが透視投影です。

### Unityでの設定

```csharp
Camera.main.orthographic = false; // 透視投影を使う
Camera.main.fieldOfView = 60f;     // 視野角（一般的には60度）
```

**Unity公式ドキュメント - Camera.orthographic：**
https://docs.unity3d.com/ja/2018.4/ScriptReference/Camera-orthographic.html

### 主な用途

・3Dアクションゲーム
・FPS（First Person Shooter）
・レースゲーム
・VRゲーム

## 3. 平行投影（Orthographic）とは

**読み方：** へいこうとうえい

**英語：** Orthographic Projection

### 特徴

平行投影は、**遠近感がない**投影方式です。

・遠くのものも近くのものも同じ大きさ
・投影線が平行に平面へ向かう
・奥行き方向（Z軸）の影響を受けない
・製図・設計図のような見え方

どんなに遠くにあっても、同じ大きさのオブジェクトは同じサイズで見えます。

### Unityでの設定

```csharp
Camera.main.orthographic = true;      // 平行投影を使う
Camera.main.orthographicSize = 5f;    // 画面に映る範囲
```

### 主な用途

・2Dゲーム
・パズルゲーム
・戦略シミュレーション（上から見下ろす視点）
・UI表示用カメラ
・ミニマップ

## 4. 開発作業での実際の使い分け

ここからが、**開発現場で実際に使うシーン**なんです。

### ケース①：マップ作成時の作業用カメラ

マップやレベルデザインをする時、**平行投影で上から見下ろす**と作業しやすいんですよね。

```csharp
// 作業用カメラの設定例
void SetupWorkCamera()
{
    Camera workCam = Camera.main;
    workCam.orthographic = true;          // 平行投影
    workCam.orthographicSize = 20f;       // 広範囲を表示
    workCam.transform.position = new Vector3(0, 50, 0);  // 上から
    workCam.transform.rotation = Quaternion.Euler(90, 0, 0); // 真下を向く
}
```

**なぜ平行投影？**

・オブジェクトの配置がしやすい
・距離感が正確に分かる
・グリッドに合わせやすい

### ケース②：UI配置用の専用カメラ

UIを配置する時は、**絶対に平行投影**です。

> カメラ構成：
> ・Main Camera（透視投影）→ ゲーム画面
> ・UI Camera（平行投影）→ UI専用

**なぜ分ける？**

・UIが3D空間の影響を受けない
・位置調整が簡単
・画面サイズが変わっても安定

### ケース③：エディタ上でSceneビューの投影を切り替え

Unityエディタの**Sceneビュー**でも投影方式を切り替えられます。

> Sceneビューの右上に「Persp」または「Iso」と表示されている
> ↓
> クリックすると切り替わる
>
> Persp（Perspective）：透視投影
> Iso（Isometric）：等角投影（平行投影の一種）

開発作業中、この切り替えをよく使うんです。

## 5. 関連用語も覚えておこう

### 投影法（とうえいほう）

**Projection Method**

物体の形・位置・寸法を平面に描く方法全般のこと。

> 投影法の種類：
> ・透視投影法（Perspective Projection）
> ・平行投影法（Orthographic Projection）

### 投影面（とうえいめん）

**Projection Plane**

物体を投影する「仮想の平面」のこと。図面やパースでいう「画面」「キャンバス」のような役割です。

Unityでは、カメラが見ている先の仮想的な平面が投影面になります。

## 6. よく使うパラメータの意味

### Field of View（FOV）

透視投影の**視野角**のこと。度数で指定します。

> 小さい値（30-40）：望遠レンズ風
> 標準値（50-60）：人間の目に近い自然な見え方
> 大きい値（70-120）：広角レンズ風

### Orthographic Size

平行投影の画面の縦方向に映る範囲の半分。

> Orthographic Size = 5
> → 縦方向に10ユニット分が画面に映る

横方向は、画面のアスペクト比によって自動計算されます。

## まとめ

**透視投影（Perspective）**
・人間の目に近い見え方
・遠近感がある
・3Dゲームに最適

**平行投影（Orthographic）**
・遠近感がない
・正確な距離・サイズが分かる
・2Dゲーム、UI、開発作業に最適
