## 【Tip 1】ショートカットをプロジェクト専用ランチャーにする

> ShortcutsはmacOS Monterey以降なら標準で使える

Shortcutsアプリで「新規ショートカット」を作成し、アイコンと名前をプロジェクト名に合わせるとSpotlightやウィジェットから狙った案件をすぐ呼び出せる。アクションは「シェルスクリプトを実行」を選んでシェルに `/bin/zsh` を指定、「入力」は「なし」のままでOK。そのまま複製してパスだけ差し替えれば、任意のプロジェクト用ランチャーがあっという間にそろう。

・Dockやメニューバーにピン留めして視覚的に整理
・「キーボードショートカットを追加」で好みのキーを割り当て
・ショートカット名に共通の接頭辞を付けるとSpotlightでヒットしやすい

## 【Tip 2】CursorとUnityを同時に呼び出すコマンド

> 1つのショートカットに2行書くだけで、任意のプロジェクトをまとめて起動できる

### Cursorを開くコマンド

プロジェクトフォルダだけ書き換えれば使い回せる。複製してパスを差し替えれば、案件ごとのCursorランチャーがすぐそろう。

```bash
open -a "/Applications/Cursor.app" "/Users/okamotoryuya/プロジェクト/Obsidian Vault"
```

### Unityを直接立ち上げるコマンド

Unity Hubを経由せず、指定したUnity本体にプロジェクトパスを渡して起動する。`ProjectName` を実際のディレクトリに、Unityのパスは自分の環境に合わせて調整する。

```bash
open -a "/Applications/Unity/Hub/Editor/2022.3.0f1/Unity.app" --args -projectPath "/Users/okamotoryuya/UnityProjects/ProjectName"
```

この2行を同じショートカットに並べておけば、「A案件のUnityとCursor」をワンボタンで呼び出せる。ウィンドウを閉じても5秒で戻れるから、作業スペースをすぐリセットできる。

## 【Tip 3】ウィジェットとショートカットキーで5秒起動を習慣化

> ❶ ウィジェットに並べる ❷ Quick Actionでメニューバー常駐 ❸ キーボード割り当てでワンボタン化

ショートカットはホーム画面ウィジェットに配置すると、デスクトップから指1本で呼び出せる。Quick Actionに登録すれば、メニューバーのコントロールセンターから即実行できる。キーボードショートカットを設定すると、UnityとCursorを同時に起動するランチャーが「1キー」で完成する。

・ショートカットのアイコンを案件ごとに色分け
・ウィジェットに主要案件を並べて視覚的に管理
・Quick Action化してメニューバーに常駐させる
・起動が早いと不要なウィンドウをこまめに閉じられる

任意のプロジェクトを5秒で呼び出せるようになると、集中が切れる前に環境を整え直せる。Macのデスクトップを毎回リセットする運用に切り替えられるので、CPU負荷やバッテリー消費も抑えやすい。
