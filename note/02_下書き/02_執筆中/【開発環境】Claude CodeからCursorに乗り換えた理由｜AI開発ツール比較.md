## Claude Codeとは

Claude Codeは、Anthropic社が提供するAIコーディングツールです。

## 乗り換えた理由

### 理由1：UIがめちゃくちゃ見やすい

これが**1番大きい**かもしれない。

Claude Codeはターミナルベースなので、出力がずらーっと流れていく。

Cursorはチャット形式で、**コードブロックやマークダウンでメリハリがある**。

しかも、コードブロックで出力されたコードを**ワンクリックでコピペできる**。

Claude Codeだとこれができなかったんですよね。

出力が見やすいし、人間が理解しやすい形になってる。

### 理由2：作業のまとめが見やすい

Cursorだと、作業が終わった後にまとめを出してもらえます。

> 表形式で変更点を整理
> 図式で構造を説明
> 差分を分かりやすく表示

プロンプトで「最後にまとめて」って言えば、きれいに整理してくれる。

### 理由3：色んなモデルが使える

Claude Codeだと、Claudeしか使えない。

Cursorは**Claude、GPT-4、Geminiなど複数のモデルが使える**。

無料で使えるモデルもあるし、どうせ制限が来るなら色んなモデル使えた方がいいですよね。

### 理由4：並列作業に対応

Claude Codeも並列作業はできたんですが、Cursorも最近対応してきました。

**複数のエージェントを立ち上げて、同時に作業できる**。

例えば、フロントエンドの修正をしながら、別のエージェントでバックエンドを書くみたいなことができる。

両方対応してるなら、UIが見やすいCursorでいいかなって。
