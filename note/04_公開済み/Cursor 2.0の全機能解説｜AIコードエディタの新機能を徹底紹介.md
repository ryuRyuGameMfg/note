## Cursor 2.0とは

Cursor 2.0は、AIを活用した次世代のコードエディタとしてリリースされた最新バージョンです。VS Codeベースのエディタでありながら、**初のエージェント型コーディングモデル「Composer」**と**マルチエージェントインターフェース**により、開発者の生産性を飛躍的に向上させます。

## 主要新機能一覧

### 機能❶：Composer（初のエージェント型コーディングモデル）

Cursor 2.0の目玉機能である**Composer**は、Cursorが独自開発した**初のエージェント型コーディングモデル**です。

**特徴：**
- 同等の知能を持つ他のモデルと比較して**4倍の速度**で動作
- 低遅延のエージェント型コーディングのために構築され、ほとんどの処理を**30秒以内**で完了
- コードベース全体のセマンティック検索を活用し、大規模なコードベースの理解と操作が向上

**起動方法：**
- Mac: `⌘I`
- Windows: `Ctrl+I`

起動すると、小さなウィンドウが表示され、「Type your instructions, / for commands」と表示されます。

**使用方法：**
1. Composerを起動（`Ctrl+I`または`⌘I`）
2. 実行したいタスクを自然言語で指示
3. 複数のファイルを選択する場合は`+`ボタンを押す
4. 編集内容は差分表示で確認できる
5. `Accept all`ボタンで変更を適用

**有効化方法：**
Cursorエディタの設定で「Beta」セクションからComposerを有効にする必要があります。

**エージェントモード：**
`⌘.`で有効化すると、コードベースの自動処理、ターミナルコマンドの実行、ファイルの作成・修正、セマンティック検索など、多彩な機能を提供します。

### 機能❷：マルチエージェントインターフェース（エージェント中心の設計）

Cursor 2.0では、**完全に新しいインターフェース**が追加されました。ファイル中心ではなく**エージェント中心**に設計されています。

**特徴：**
- 開発者は**望む結果に集中**し、エージェントが詳細を処理
- コードに深く入り込む必要がある場合は、クラシックなIDEに簡単に切り替え可能
- **複数のエージェントを並行して実行**でき、互いに干渉しない
- Git worktreesやリモートマシンを活用して実現
- **複数のモデルに同じ問題を試させ、最良の結果を選択**することで、特に難しいタスクにおいて最終的な出力の品質が大幅に向上

### 機能❸：エディタ内ブラウザ（Embedded Browser）

ブラウザがエディタ内に組み込まれ、開発作業をより効率的に進められます。

**特徴：**
- エディタ内で直接ブラウジング可能
- **要素を選択する強力なツール**が利用可能
- **フルdev tools**が利用可能
- **MCPコントロール**でエージェントがブラウザを操作可能

### 機能❹：改善されたコードレビュー

エージェントが行った変更を、**コードベース全体で1箇所にまとめて確認**できるようになりました。

**特徴：**
- 異なるファイル間を移動する必要がなく、全ての変更を1箇所で確認可能
- エージェントが行った変更の全体像を把握しやすい

### 機能❺：音声入力（Voice Mode）

ネイティブな音声からテキストへの変換機能が追加されました。

**特徴：**
- キーボードを使わずに音声でコードやコメントを入力可能
- 手が離せない状況でも作業を続けられる

### 機能❻：その他の改善点

**言語サーバーのパフォーマンス向上：**
- アプリ全体で言語サーバーのパフォーマンスが向上
- テキストレンダリングの改善

**ルール＆コマンドのディープリンク：**
- ルールやコマンドへのディープリンクが可能
- ワークフローの効率化

**バックグラウンドエージェントのプランモード：**
- バックグラウンドエージェントのプランモードが追加
- より効率的なタスク管理が可能

**エンタープライズ向けセキュリティコントロール：**
- エンタープライズ向けの新しいセキュリティコントロールが追加
- 企業での利用がより安全に

## 注意点・デメリット

### 注意点❶：有料プランが必要

Composerを使用するには、**Cursor Proプラン（月額20ドル）以上**の契約が必要です。

**料金プラン：**
- Hobbyプラン：無料（基本的なAI機能を試せる、Pro版機能の2週間試用、2000回のコード補完、50回の遅いプレミアムリクエスト）
- Proプラン：月額20ドル（無制限のコード補完、月500回の高速プレミアムリクエスト、高度なAI機能）
- Businessプラン：ユーザーあたり月額40ドル（企業やチーム向け、組織全体でのプライバシーモード強制、チームの一括請求、管理者用ダッシュボード、SSOサポート）

### 注意点❷：ベータ版である

Composerは現在ベータ版であり、機能やインターフェースが今後変更される可能性があります。

### 注意点❸：学習コスト

最初の1週間は、**どのように指示すればいいか**迷うことがあります。従来のVS Codeのワークフローから切り替えるには、慣れが必要です。

### 注意点❹：完璧ではない

複雑なロジックになると、**期待通りの結果が得られない**こともあります。ただし、従来のAIツールと比べると、精度は大幅に向上しています。

**推奨：**
- 生成されたコードは必ずレビューする
- 重要な部分は手動で確認する
- 差分ビューで変更内容を確認してから適用する

## まとめ

Cursor 2.0は、以下の革新的な機能を提供します：

- **Composer**：初のエージェント型コーディングモデル、4倍の速度でコード生成、30秒以内で完了、大規模コードベースの理解と操作が向上
- **マルチエージェントインターフェース**：エージェント中心の設計、複数のエージェントを並行実行し、最良の結果を選択可能、Git worktreesやリモートマシンを活用
- **エディタ内ブラウザ**：要素選択ツール、フルdev tools、MCPコントロール
- **改善されたコードレビュー**：エージェントの変更を1箇所で確認可能
- **音声入力**：ネイティブな音声からテキストへの変換
- **その他の改善**：言語サーバーのパフォーマンス向上、ルール＆コマンドのディープリンク、バックグラウンドエージェントのプランモード、エンタープライズ向けセキュリティコントロール

これらの機能により、開発効率が大幅に向上し、開発者は**コードの本質的な部分に集中**できるようになります。

有料プランが必要という点はありますが、**開発効率の向上を考えると、投資価値は十分にあります**。
