# Roblox開発をMacOSで始める方法（調査結果）

## 調査日
2025年1月

---

## 📋 基本情報

### Roblox Studioとは
- Robloxの公式ゲーム開発ツール
- Windows/MacOS両対応
- 無料で利用可能
- Lua言語でスクリプト開発

---

## 🔧 インストール方法

### 方法1：公式サイトから直接ダウンロード（推奨）
```
1. https://create.roblox.com/ にアクセス
2. 「制作を開始」ボタンをクリック
3. Robloxアカウントにログイン（または新規作成）
4. RobloxStudio.dmg がダウンロードされる
5. DMGファイルを開き、アプリケーションをインストール
```

### 方法2：Roblox公式サイト経由
```
1. https://www.roblox.com/ にアクセス
2. ログイン後、任意のバーチャル空間にアクセス
3. 右上のメニューアイコン → 「編集」を選択
4. Roblox Studioのインストールが促される
5. 指示に従ってインストール
```

## 💻 システム要件

### 最低要件
- **OS**: macOS 10.13（High Sierra）以降
- **CPU**: 1.6 GHz以上
- **RAM**: 4 GB以上
- **ストレージ**: インストール後に最低10 GBの空き容量
- **インターネット**: 4〜8 Mbps以上の速度

### 推奨要件
- **グラフィックス（デスクトップ）**: 5年以内に製造された専用GPU搭載モデル
- **グラフィックス（ノートPC）**: 3年以内に製造された統合GPU搭載モデル

### Apple Silicon（M1/M2/M3）対応
- **対応状況**: 確認が必要（検索結果では明確な記載なし）
- **推奨**: 公式サイトで最新情報を確認
- **注意**: Rosetta 2での動作の可能性あり

## ⚠️ 注意点・トラブルシューティング

### セキュリティ
- **公式サイトからのみダウンロード**: 非公式サイトからのダウンロードはセキュリティリスクあり
- **公式サイト**: https://create.roblox.com/ または https://www.roblox.com/

### 日本語対応
- Roblox Studioは日本語に対応している
- 設定によっては英語表示になる場合あり
- 必要に応じて言語設定を確認

### よくある問題
1. **インストールできない**
   - macOSのバージョンが10.13以降か確認
   - セキュリティ設定でダウンロードしたアプリの実行を許可

2. **動作が重い**
   - システム要件を確認
   - グラフィックス設定を調整
   - 他のアプリケーションを終了

3. **インターネット接続エラー**
   - 最低4〜8 Mbpsの速度を確保
   - ファイアウォール設定を確認

## 🛠️ 開発環境のセットアップ

### 基本的な開発フロー
```
1. Roblox Studioを起動
2. 新しいプロジェクトを作成
3. 3Dモデルを配置（パーツ、モデルなど）
4. Luaスクリプトで動作を実装
5. プレイテストで動作確認
6. 公開設定でゲームを公開
```

### Lua言語について
- Roblox StudioではLua言語を使用
- スクリプトエディタ内でコーディング可能
- 外部エディタ（VS Codeなど）との連携も可能
