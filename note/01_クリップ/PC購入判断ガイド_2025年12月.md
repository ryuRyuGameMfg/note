# PC購入判断ガイド（2025年12月版）

## 🎯 結論：今買うべきか、来年待つべきか

### **推奨：今買う（ただし最高モデルを）**

**理由：**
1. **価格は今後さらに上昇する見込み**
   - 2026年初頭にGPU価格がさらに上昇予測
   - メモリ価格も2026年中頃まで高騰が続く見込み
   - 現在の価格が「安い」とは言えないが、今後はさらに高くなる可能性が高い

2. **電気料金も上昇する**
   - 2026年度から東京電力エリアで8-10%の電気料金増加
   - AI需要による電力不足が懸念される
   - 高性能GPUは消費電力も高いため、ランニングコストも考慮が必要

3. **最高モデルを買うべき理由**
   - VRAMは後から追加できない（GPU交換が必要）
   - 将来のローカルLLMに対応するには、今のうちに大容量VRAMを確保すべき
   - 価格上昇を考えると、今買って長く使う方がコストパフォーマンスが良い

---

## 💾 PCパーツの追加・アップグレード可能性

| パーツ | 追加可能？ | 備考 |
|--------|----------|------|
| **メモリ（RAM）** | ✅ **可能** | デスクトップPCは増設可能。ノートPCは機種による。マザーボードの最大容量を確認。 |
| **VRAM（グラボメモリ）** | ❌ **不可能** | GPUに内蔵されているため、増やすにはGPU交換が必要。**最重要ポイント** |
| **ストレージ（SSD/HDD）** | ✅ **可能** | デスクトップPCは追加可能。ノートPCもM.2スロットがあれば追加可能。 |

### ⚠️ 重要なポイント

**VRAMは後から追加できないため、最初から大容量を選ぶべき**

- RTX 5090（32GB VRAM）を選べば、将来の大規模LLMにも対応可能
- RTX 5070（12GB）やRTX 5080（16GB）では、将来的に不足する可能性が高い

## 🎮 RTX「Ti」バージョンと通常版の違い

### RTX 5070 vs RTX 5070 Ti 比較

| 項目 | RTX 5070 | RTX 5070 Ti | 差 |
|------|----------|-------------|-----|
| **CUDAコア数** | 6,144 | 8,960 | +46% |
| **VRAM** | 12GB GDDR7 | **16GB GDDR7** | +4GB |
| **メモリ帯域幅** | 692 GB/s | 896 GB/s | +29% |
| **消費電力** | 250W | 300W | +50W |
| **価格** | $549（約9万円） | $749（約12万円） | +約3万円 |

### Tiバージョンの特徴

- **Ti = Titanium（チタン）**の略で、高性能版を意味する
- 通常版より**CUDAコア数が多い** = 処理速度が速い
- **VRAM容量が多い** = より大規模なモデルに対応可能
- **価格が高い** = 約20-30%高価

### 推奨

- **ローカルLLM重視**: RTX 5070 Ti（16GB VRAM）を推奨
- **ゲーム重視**: RTX 5070（12GB VRAM）でも十分な場合が多い

## 📈 ローカルLLMの将来性：オンラインLLMに追いつくか？

### 現状（2025年12月）

| 項目 | オンラインLLM | ローカルLLM | 差 |
|------|-------------|-----------|-----|
| **最新モデル** | GPT-5.2, Claude Opus 4.5 | Qwen2.5 72B, DeepSeek R1 | 約2-3世代遅れ |
| **MMLU（知識理解）** | 85-89% | 85-88% | ほぼ同等 |
| **コーディング能力** | 75-81% | 85-87% | **ローカルLLMが上回る** |
| **数学推論** | 93-100% | 87-92% | オンラインLLMが優位 |

### 将来予測（2026-2027年）

#### ✅ 追いつく可能性が高い理由

1. **ハードウェアの進化**
   - RTX 5090は前世代比200%の性能向上
   - 2027年までにNPUが100TOPS以上に進化
   - 7BパラメータクラスをGPUなしで実行可能に

2. **ソフトウェア最適化**
   - 1-bit量子化の実用化（2025-2026年）
     - VRAM使用量を1/16に削減
     - 性能劣化を5%以下に抑制
   - Speculative Decodingの普及
     - 推論速度が3-5倍向上
     - リアルタイム対話が可能に

3. **市場成長**
   - 2025年：1,200億円
   - 2026年：2,800億円（+133%）
   - 2027年：5,600億円（+100%）
   - 急成長により開発が加速

4. **コスト削減**
   - 2025年下半期：20-30%削減
   - 2026年：さらに30-40%削減

#### 📊 予測タイムライン

| 時期 | ローカルLLMの状況 |
|------|----------------|
| **2025年12月（現在）** | GPT-4o / Claude 3.7レベルに到達 |
| **2026年中頃** | GPT-5.2 / Claude Opus 4.5レベルに接近 |
| **2027年** | オンラインLLMと同等レベルに到達 |

### 結論

**ローカルLLMは2026-2027年にかけてオンラインLLMに追いつく可能性が高い**

- ハードウェア・ソフトウェアの両面で急速に進化
- 特にコーディング能力では既にオンラインLLMを上回る
- プライバシー・セキュリティ面での優位性も大きい

## 🎯 最終推奨：Alienware vs 自作PC

**推奨：自作PC**

**理由：**
1. **VRAMを最大限確保できる**
   - RTX 5090（32GB VRAM）を選択可能
   - AlienwareではRTX 5090対応モデルが限定的

2. **コストパフォーマンス**
   - 同じスペックで20-30%安く済む
   - 節約分でメモリやストレージを増強可能

3. **将来のアップグレード**
   - 自由にパーツ交換可能
   - VRAM不足時のGPU交換が容易

4. **カスタマイズ性**
   - 用途に合わせて最適化可能
   - 冷却システムも自由に選択可能

### Alienwareを選ぶ場合

**条件：**
- デザイン性を最優先
- サポートが必要
- 組み立てが面倒
- 予算に余裕がある

**推奨モデル：**
- **デスクトップ**: Alienware Area-51 + RTX 5080（約57万円〜）
- **ノートPC**: Alienware 16 Area-51 + RTX 5080 Laptop（約65万円）
