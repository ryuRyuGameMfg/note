# 目的
記事タイプ（レビュー/ノウハウ/実績）ごとの骨子を生成する。

# 入力
- 記事タイプ
- ターゲット/悩み
- 実績・数値
- CTA文言（任意）

# 出力形式
- 見出し（##, ###まで）
- 各節の要点（3–5点）
- CTA配置（導入直後/末尾）

# 制約
- 2500–4000字想定の骨子
- 引用ブロック内装飾なし
- URLは独立行

# プロンプト
「{記事タイプ}」の記事骨子を作成。
ペルソナ: {ターゲット/悩み}
使える実績: {実績・数値}
要件:
- 基本構造（フック/問題/解決/結果/まとめ）
- 見出しは2階層まで
- デメリットも明記
- CTAは導入直後と末尾に配置（文言: {CTA文言}）
出力は「見出し＋箇条書きの要点」のみ。
