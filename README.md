# note & X 制作ワークスペース

note記事・X投稿の制作・管理用リポジトリ。

## クイックスタート

### note記事

1. アイデア作成: `note/01_アイデア/`
2. 執筆: `note/02_下書き/02_執筆中/`
3. 公開準備: `note/03_公開準備完了/`
4. 公開後: `note/04_公開済み/`
5. 分析: `note/05_分析/`

### X投稿

1. アイデア作成: `X/01_アイデア/`
2. 下書き: `X/02_下書き/`
3. 公開準備: `X/03_公開準備完了/`
4. 公開後: `X/04_公開済み/`
5. 分析: `X/05_分析/`

## 運用ルール

`運用ルール/` フォルダに執筆・運用ガイドを集約：

| ファイル | 内容 |
|---------|------|
| `01_執筆ガイド.md` | 記事・投稿の書き方ルール |
| `02_チェックリスト.md` | 公開前の確認項目 |
| `03_テンプレート.md` | 記事・投稿のテンプレート |
| `04_AI運用.md` | AI活用時のプロンプト・フロー |
| `05_X運用スタイル.md` | X投稿の口調・運用スタイル |
| `06_CTAパターン.md` | 記事別CTAパターン集 |

## ディレクトリ構成

```
.

│   ├── 01_執筆ガイド.md
│   ├── 02_チェックリスト.md
│   ├── 03_テンプレート.md
│   ├── 04_AI運用.md
│   ├── 05_X運用スタイル.md
│   └── 06_CTAパターン.md
├── note/               # note記事管理
│   ├── 01_アイデア/
│   ├── 02_下書き/
│   ├── 03_公開準備完了/
│   ├── 04_公開済み/
│   └── 05_分析/
├── X/                  # X投稿管理
│   ├── 01_アイデア/
│   ├── 02_下書き/
│   ├── 03_公開準備完了/

```

## 運用ルール（要点）

- **1記事=1ファイル**: 段階移行時は前段階を削除して移動
- **URL**: 独立行に配置
- **引用ブロック**: プレーンテキストのみ
- **CTA**: 導入直後/末尾の2箇所、統一フォーマット

## 配布用フォルダ（export/）

外部環境で使用するためのスキル・テンプレートを格納。
