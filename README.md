# note・X 制作ワークスペース

ゲーム開発所RYURYUのnote記事・X投稿を管理するワークスペース。

## 構成

```
note/（note記事の管理）
X/（X投稿の管理）
運用ルール/（執筆・運用のガイドライン）
```

## 運用ルール

記事・投稿を作成する際のガイドラインは 運用ルール/ フォルダにある。

```
運用ルール/
├── 00_はじめに.md（最初に読む。読む順番を案内）
├── 01_運用方針/（コンセプト・ターゲット・KPI）
├── 02_note/（note記事の書き方・テンプレート）
├── 03_X/（X投稿の書き方・テンプレート）
├── 04_SEO/（公式アルゴリズム情報）
└── 05_共通/（CTA・AI運用）
```

## AIで記事を作成する場合

1. 運用ルール/00_はじめに.md を最初に読む
2. そこに書いてある順番でドキュメントを参照
3. テンプレートに沿って作成
4. チェックリストで確認

## 記事ライフサイクル

アイデア → 執筆中 → 公開準備完了 → 公開 → Xで告知 → 分析
