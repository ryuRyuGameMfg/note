# note

note記事・X投稿の制作ワークスペース。

## フォルダ構成

```
note/           note記事管理（アイデア→下書き→公開）
X/              X投稿管理
NotePublisher/  noteリリースツール（Next.js）
DOCS/           運用ルール・ガイド
AI_PROJECT_KIT/ AI開発プロジェクト用テンプレート集
export/         エクスポート用
```

## ツール

### NotePublisher

記事の公開作業を効率化するローカルツール。

```bash
cd NotePublisher
npm run dev
# http://localhost:3010
```

## AI向け

Claude Code等のAIツールを使用する場合は `CLAUDE.md` を参照。
