# noteチェックリスト

参照: 04_SEO/公式アルゴリズム情報.md

## 公式アルゴリズム対応（最優先）

- [ ] 冒頭3行に記事の要約（自己紹介・広告NG）
- [ ] ハッシュタグは3〜5個
- [ ] 見出し画像は1280×670px
- [ ] 公開直後にXで告知する準備

## 内容・構成

- [ ] 2500–4000字に収まっている
- [ ] 基本構造（フック/問題/解決/結果/まとめ）
- [ ] 権威性を数字で提示（件数/期間/金額/頻度）
- [ ] デメリット・注意点を明記
- [ ] 具体例・体験談を含む

## 視覚・note最適化

- [ ] 見出しは##/###のみ（2階層まで）
- [ ] 引用ブロック内に装飾なし（太字/斜体禁止）
- [ ] 引用内の数字箇条書き（1. 2. 3.）を未使用
- [ ] 空の引用行（>のみ）なし
- [ ] URLは独立行（前に説明文）

## CTA・導線

- [ ] 導入直後（冒頭3行の後）にCTA
- [ ] 末尾にもCTA
- [ ] 追伸なし（CTAで終了）

## コピペ安定性

- [ ] 水平線（---）未使用または必要最小限
- [ ] 引用直後にコードブロックを置かない

## SEO対策

- [ ] タイトルにメインKW＋具体数字（期間/金額/件数）
- [ ] タイトルは省略されない長さ（25文字目安）
- [ ] キーワードはタイトル前半に配置
- [ ] 見出しにサブKWを散りばめる
- [ ] 内部リンク（関連note）を2–3本
- [ ] 検索意図に対する答えが明確

## AEO対策（AI検索最適化）

- [ ] Q&A/FAQを2–5問用意
- [ ] 定義・手順・比較は箇条書き短文で要約可能に
- [ ] 固有名詞を正式名で明記
- [ ] 結論先出しの要約を100–200字で用意

## 公開後チェック

- [ ] Xで告知した（スキ10個目標）
- [ ] Xの告知投稿に自分でURLを返信した
- [ ] コメントが来たら返信
- [ ] 1週間後にPV/スキ数を確認

## クイックチェック（最低限5項目）

- [ ] 冒頭3行に要約（自己紹介・広告NG）
- [ ] ハッシュタグ3〜5個
- [ ] 2500-4000字
- [ ] CTA2箇所（導入後・末尾）
- [ ] 公開後Xで告知
