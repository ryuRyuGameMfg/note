# X執筆ガイド

参照: 04_SEO/公式アルゴリズム情報.md

## 投稿ライフサイクル

アイデア → 下書き → 公開準備完了 → 投稿 → 返信対応 → 分析

原則:
- 宣伝2：日常8の比率を守る
- 返信には必ず返信する（アルゴリズム最重要）
- 外部リンクは本文に入れない

## 公式アルゴリズム対応

| 対応事項 | 理由 | 優先度 |
|---------|------|--------|
| 質問で締める | 返信=いいねの27倍の価値 | 必須 |
| 返信には必ず返信する | 返信への返信=いいねの150倍の価値 | 必須 |
| 外部リンクは本文に入れない | 外部リンクは低評価される | 必須 |
| リンクは返信欄に配置 | 「リプ欄に」と明記 | 必須 |
| 画像・動画を添付 | 1.7倍拡散効果 | 推奨 |

エンゲージメント重み付け:

| アクション | 重み（いいね=1） | 対策 |
|-----------|-----------------|------|
| いいね | 1 | - |
| リツイート | 2 | 保存したくなる内容 |
| 返信 | 27 | 質問で締める |
| 返信への返信 | 150 | 必ず返信する |

## 基本構成

1. フック（1行目で目を引く）
2. 本文（具体的な内容・価値）
3. 質問で締める（返信を促す）
4. ハッシュタグ（2-4個）

## 外部リンクの配置

外部リンクを本文に入れると低評価される。リンクは返信欄に配置し、本文には「詳細はリプ欄に」「プロフから」と明記する。

正しい配置:
```
【本文】
〇〇について書きました！

・ポイント1
・ポイント2
・ポイント3

皆さんは〇〇どうしてますか？
詳細はリプ欄に

#Unity #ゲーム開発

【返信欄（自分で投稿）】
記事はこちら：
https://note.com/xxx/n/xxx
```

悪い配置:
```
〇〇について書きました！
https://note.com/xxx/n/xxx  ← 本文にURL入れると低評価
```

## トーン

基本トーン: カジュアルで親しみやすい、友達に話すような口調

使う表現:
- 「〜なんですよね」「〜なんです」
- 「めちゃくちゃ〜」「マジで〜」
- 「やったー！」「おかげさまで」
- 「ぜひ〜してみて」

避ける表現:
- 「〜いたします」「〜でございます」（硬すぎ）
- 「ご検討ください」（営業っぽい）
- 「是非ともご利用を」（宣伝臭い）

一人称:
- 「僕」「俺」（カジュアルな場面）
- 「私」（フォーマルな場面）

## 効果的な投稿パターン

質問型（返信獲得狙い・推奨）:
```
今日の開発メモ。

〇〇の機能、ようやく動いた！
△△で苦戦したけど解決。

皆さんは〇〇で困った経験ありますか？

#Unity #ゲーム開発
```

失敗談型（共感獲得狙い）:
```
正直に言う。

最初の見積もり、甘すぎた。
時給換算したらバイト以下だった...

今は〇〇を意識してる。

同じ経験ある人いますか？

#個人開発 #フリーランス
```

Tips型（保存獲得狙い）:
```
個人開発で大事だと思うこと。

・〇〇する
・△△を意識
・□□を避ける

保存しておくと便利かも

#個人開発 #Unity
```

## ハッシュタグ

常用（2-3個選択）:
- #Unity
- #ゲーム開発
- #個人開発
- #フリーランス

状況別:

| 状況 | タグ |
|------|------|
| AI関連 | #AI #ChatGPT |
| 副業系 | #ココナラ #副業 |
| 進捗 | #開発進捗 #今日の積み上げ |
| note告知 | #note |

ルール:
- 2-4個が適切
- 多すぎるとスパムっぽい
- 本文末尾に配置

## 投稿タイミング

| 時間帯 | 優先度 | 理由 |
|--------|--------|------|
| 平日 20:00-22:00 | 高 | 帰宅後のゴールデンタイム |
| 土曜 10:00-12:00 | 中 | 朝のゆっくり時間 |
| 平日 12:00-13:00 | 中 | 昼休み |

頻度:
- 週2〜3回を目安
- 無理に毎日投稿しない
- 質 > 量

## 投稿後の対応

返信への返信 = いいねの150倍の価値。来た返信には必ず返信し、会話が続くとアルゴリズムが高評価する。

投稿後フロー:
1. 投稿
2. 来た返信に返信する（必須）
3. 会話を続ける
4. エンゲージメント確認
5. 伸びた/伸びなかった理由を分析

## やってはいけないこと

- 本文に外部リンクを入れる
- URLのみの投稿
- 宣伝ばかりの投稿
- 返信を無視する
- 大量フォロー/アンフォロー
- 攻撃的・炎上しそうな発言
