# Xチェックリスト

参照: 04_SEO/公式アルゴリズム情報.md

## 公式アルゴリズム対応（最優先）

- [ ] 質問で締める（返信を促す）
- [ ] 外部リンクは本文に入れていない
- [ ] リンクを入れる場合は返信欄に
- [ ] 画像または動画を添付

## 内容

- [ ] 1行目でフック（目を引く）
- [ ] 具体的な価値・メリットを記載
- [ ] 数字を含む（具体性）
- [ ] 攻撃的・炎上しそうな内容でない

## 形式

- [ ] 文字数は適切か（280字以内 or スレッド形式）
- [ ] ハッシュタグは2-4個
- [ ] 改行で読みやすく

## タイミング

- [ ] 投稿時間は適切か
  - 平日夜 20:00-22:00（ゴールデンタイム）
  - 土曜 10:00-12:00
  - 平日昼 12:00-13:00

## 投稿後（必須）

- [ ] 来た返信には必ず返信する（返信への返信=150倍の価値）
- [ ] インプレッション/エンゲージメント確認
- [ ] 伸びた投稿の共通点を分析

## note記事告知時の追加チェック

リンク配置:
- [ ] 本文にURLを入れていない
- [ ] 「リプ欄に」「プロフから」と明記
- [ ] 投稿後すぐに返信欄にURLを投稿

告知文の内容:
- [ ] 記事の要点を3つ程度記載
- [ ] 読むメリットが明確
- [ ] 質問で締めて返信を促す

## サービス告知時の追加チェック

- [ ] 本文にURLを入れていない
- [ ] 「リプ欄に」「プロフから」と明記
- [ ] 宣伝と日常の比率は2:8を維持
- [ ] 押し付けがましくない

## クイックチェック（最低限4項目）

- [ ] 質問で締める
- [ ] リンクは返信欄
- [ ] 画像添付
- [ ] 返信には返信

## やってはいけないことチェック

- [ ] 本文に外部リンクを入れていない
- [ ] 宣伝ばかりになっていない
- [ ] 攻撃的な内容でない
- [ ] URLのみの投稿でない

## 週次チェック

- [ ] インプレッション/エンゲージメント確認
- [ ] 伸びた投稿の共通点分析
- [ ] 返信への対応漏れチェック
- [ ] 宣伝と日常の比率は2:8を維持

## 月次チェック

- [ ] フォロワー増減
- [ ] 最も伸びた投稿TOP3
- [ ] note/ココナラへの流入数
- [ ] 成功パターンの蓄積
