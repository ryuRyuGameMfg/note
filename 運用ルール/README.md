# 運用ルール

このフォルダには、note記事・X投稿の執筆・公開・分析に関する運用ルールをまとめています。

## フォルダ構成

```
運用ルール/
├── README.md（このファイル）
├── 01_執筆ガイド.md        # 記事・投稿の書き方ルール
├── 02_チェックリスト.md    # 公開前の確認項目
├── 03_テンプレート.md      # 記事・投稿のテンプレート
├── 04_AI運用.md            # AI活用時のプロンプト・フロー
└── 05_X運用スタイル.md     # X投稿の口調・運用スタイル
```

## 使い方

1. **執筆前**: `01_執筆ガイド.md` でルールを確認
2. **公開前**: `02_チェックリスト.md` で品質チェック
3. **テンプレ活用**: `03_テンプレート.md` から骨子をコピー
4. **AI活用時**: `04_AI運用.md` のプロンプトを使用
5. **X投稿時**: `05_X運用スタイル.md` で口調・スタイルを確認
