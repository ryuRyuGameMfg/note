> 音声入力：約5分

## プロが使う音声入力テクニック3選

### 1️⃣ 句読点の自動挿入

**音声コマンド：**
> 「まる」→ 。
> 「てん」→ 、
> 「かいぎょう」→ 改行
> 「かっこ」→ 「」
> 「びっくり」→ ！
> 「はてな」→ ？

**実際の使用例：**
> 音声：「今日は天気がいいですね まる
> 　　　 明日も晴れるといいな びっくり」

> 結果：「今日は天気がいいですね。
> 　　　 明日も晴れるといいな！」

**プロのコツ：**
自然に話しながら句読点を入れる練習をすると、
1週間で慣れて、通常の会話のように使えるようになります。

### 2️⃣ 専門用語の辞書登録

**よく誤変換される専門用語：**
> 「ジェイソン」→ JSON ×Jason
> 「エーピーアイ」→ API ×エーピーアイ
> 「ユニティ」→ Unity ×ユニティー
> 「リアクト」→ React ×リアクト

**Windows辞書登録：**
```
❶ IMEツールバー → ツール → 単語の登録
❷ 単語：「JSON」 読み：「じぇいそん」
❸ 品詞：「名詞」で登録
```

**Mac辞書登録：**
```
❶ システム設定 → キーボード → テキスト入力
❷ 「＋」ボタンで新規追加
❸ 入力：「じぇいそん」 変換：「JSON」
```

**効果：**
- 専門用語の誤変換がゼロになる
- 話すスピードを落とさずに正確な入力
- ストレスフリーな音声入力体験

### 3️⃣ ショートカットとの組み合わせ

**音声入力 + キーボードショートカット：**
> **Windows:**
> Win + H → 音声入力開始
> Ctrl + S → 保存（音声：「コントロール エス」）
> Ctrl + Z → 元に戻す（音声：「コントロール ゼット」）

> **Mac:**
> Fn × 2 → 音声入力開始
> Cmd + S → 保存（音声：「コマンド エス」）
> Cmd + Z → 元に戻す（音声：「コマンド ゼット」）

**実践例：**
> ❶ 音声で文章を入力
> ❷ 「コマンド エー」と言って全選択
> ❸ 「コマンド シー」でコピー
> ❹ 別のアプリに移動
> ❺ 「コマンド ブイ」でペースト

> → キーボードに一切触れずに操作完了

## 音声入力で起きる問題と解決法

### 問題1：周りがうるさい環境

**解決策：ノイズキャンセリング**
```
Windows:
設定 → システム → サウンド → 音声入力
→ 「ノイズ抑制」をON

Mac:
システム設定 → サウンドとビデオ → 音声分離
→ 「標準」または「音声分離」を選択
```

**物理的対策：**
- 指向性マイクの使用
- ヘッドセットマイクの活用
- 静かな場所への移動

**最初は誰もが恥ずかしい：**
> 初日：「なんか独り言言ってる人みたい...」

**克服方法：**
❶｜最初は自宅で練習
❷｜イヤホンマイクで小声でもOK
❸｜電話してるフリをする
❹｜慣れたらどこでもできる

### 問題3：方言や訛り

**音声認識の学習機能：**
> **Windows音声認識：**
> 「音声認識のトレーニング」を実行

> **Mac音声入力：**
> 使えば使うほど自動で学習
> → 2週間で方言も認識

**コツ：**
- 最初は標準語を意識
