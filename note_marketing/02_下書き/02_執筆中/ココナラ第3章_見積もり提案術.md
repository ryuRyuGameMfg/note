## 最初はひどい見積もりを送ってた

恥ずかしいけど、最初の見積もりを晒す。

```

```

3行で終わり。
根拠もなければ、熱意もない。

本気で分からなかった。

## 売れてる人の提案文を100個分析した

ココナラには**公開依頼**という機能がある。
過去のやり取りが公開されてる。

これを片っ端から読んだ。
**100個以上。**

### 発見した共通パターン

売れてる人の提案には、**型**があった。

```
1. 挨拶・共感
2. 実現可能の明言
3. 内容の整理・確認
4. 費用の内訳
5. スケジュール
6. 実績の提示
7. 熱意のあるクロージング
```

**全部入ってる提案は、ほぼ受注してた。**

### ダメな提案の特徴

逆に、受注できてない提案：

- 金額だけ
- 「できます」だけ
- コピペ感満載
- 相手の要望を理解してない

**要は、雑。**

### このテンプレートがなぜ効くのか

**❶ 最初に「できる」と明言**
不安を取り除く

**❷ 内容を整理して見せる**

**❸ 内訳を明確にする**
不透明な部分をなくす

**❹ 実績を3つ見せる**
多すぎず少なすぎず、ちょうどいい

**❺ 追加提案で価値を演出**
「お得感」を感じてもらう

**❻ 個人的な感想を入れる**
テンプレじゃない、あなただけへの提案

## 返信速度が受注率に直結する

データ取ってみて分かった。

```
返信時間と受注率：
〜1時間：受注率75%
〜3時間：受注率60%
〜6時間：受注率40%
翌日以降：受注率15%
```

**1時間以内の返信で、受注率5倍。**

相手は複数人に相談してる。
**最初に返信した人が有利。**

## 断られた時の分析が成長につながる

断られたら、必ず理由を聞く。

```
「今回は見送らせていただきます」

↓

「承知いたしました。
差し支えなければ、今後の参考のため
見送りの理由を教えていただけませんか？」
```

- 予算オーバー（40%）
- 納期が合わない（25%）

**この分析が、次の改善につながる。**


相談数は1.5倍だけど、
