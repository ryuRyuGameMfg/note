## 【第1章】最初の1件を獲得するまでの泥臭い努力

2022年4月、ココナラに登録した。

Unity開発サービスを5,000円で出品。
競合は1,000円〜3,000円がほとんどだったけど、
「安売りしたくない」って変なプライドがあった。

**結果：1ヶ月間、問い合わせゼロ。**

> 「やっぱり実績ゼロじゃ無理か...」

でも諦めなかった。
毎日3つのことを徹底した。

### ❶ プロフィール入力率100%

```
✓ 本人確認書類提出
✓ NDA（機密保持契約）締結
✓ スキル/経歴を全項目入力
✓ ポートフォリオ10個以上登録
```

特に**NDA締結**が効いた。
企業案件の問い合わせが増えた。

### ❷ 競合上位10サービスを分析

毎日1時間、売れてるサービスを研究した。

```
分析項目：
- タイトルの共通点
- 価格設定の理由
- サービス説明の構成
- 購入者の評価コメント
```

**発見：みんな「納期」を明記してなかった。**

### ❸ 破格の初回限定価格

プライド捨てた。

```
通常5,000円 → 初回限定500円
```

**赤字覚悟。でも最初の1件が欲しかった。**

5月15日、初めての問い合わせが来た。
「Unityでシューティングゲーム作れますか？」

即レスで「はい！できます！」

初案件は**40時間かけて500円**。
時給12.5円。バイトの方が100倍マシ。

でも、**評価5.0をもらえた**。

これが全ての始まりだった。

## 【第5章】月70万円を実現した戦略的価格設定

最初は500円スタート。
今は平均単価**5万円**。

どうやって100倍にしたか？

### 価格を上げるタイミング

```
評価10件達成 → 500円 → 3,000円
評価30件達成 → 3,000円 → 8,000円
評価50件達成 → 8,000円 → 15,000円
評価100件達成 → 15,000円 → 30,000円
カテゴリ1位 → 30,000円 → 50,000円
```

**実績が価格を正当化する。**

### 松竹梅の法則

3つの価格帯を用意。

```
🥉 ライトプラン：15,000円
　 簡単な機能追加・修正

🥈 スタンダード：50,000円 ← 7割がこれ選ぶ
　 ミニゲーム開発

🥇 プレミアム：150,000円
　 本格アプリ開発
```

**真ん中が売れる。これが心理学。**

### 時間単価を意識する

```
初期：40時間で500円 = 時給12.5円
現在：10時間で50,000円 = 時給5,000円
```

効率化の積み重ねが、**時給400倍**を実現。

### 断る勇気

安い案件は断る。

```
「予算5,000円でRPG作れますか？」
→ 「申し訳ございません。
　　そのご予算では対応が難しいです。
　　もしご予算の調整が可能でしたら、
　　改めてご相談ください。」
```

**安請け合い = 自分の価値を下げる**

### 実際の月次売上推移

