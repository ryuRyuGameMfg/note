2022年4月。
大学3年生だった僕は、ココナラに登録した。

Unity開発を独学で2年。
「そろそろお金稼げるんじゃない？」って軽い気持ちだった。

**現実は甘くなかった。**

最初の1ヶ月、売上ゼロ。
問い合わせすらゼロ。

> 「やっぱ学生じゃ無理か...」

何度も諦めかけた。
でも今は**月70万稼げる**ようになった。

最初の1件を取るまでにやった、**泥臭い努力**を全部書く。

## プライドが邪魔をした最初の1ヶ月

Unity開発サービスを**5,000円**で出品した。

競合を見ると、1,000円〜3,000円がほとんど。
でも「安売りしたくない」って変なプライドがあった。

> 「技術に自信あるし、5,000円は妥当でしょ」

**大間違いだった。**

実績ゼロの学生に、5,000円払う人なんていない。
当たり前だ。

1週間経過：**閲覧数3**
2週間経過：**閲覧数8**
1ヶ月経過：**問い合わせ0**

サービスページすら見てもらえない。
**存在してないのと同じ**だった。

## プロフィールを100%埋めたら変化が起きた

ある日、売れてる人のプロフィールを研究してて気づいた。

**みんな、プロフィール埋まってる。**

僕のプロフィール入力率：**30%**
売れてる人：**100%**

違いは明白だった。

### 本人確認でココナラからの信頼を得る

まず**本人確認書類**を提出。
運転免許証の写真撮って、アップロード。

すると、プロフィールに**本人確認済みバッジ**が付いた。

```
変化：
閲覧数 8 → 15（翌週）
```

たった7増えただけ。
でも**初めての変化**だった。

### NDA締結で企業案件を狙う

**NDA（機密保持契約）**も締結した。

正直、学生には関係ないと思ってた。
でもこれが**大きな差別化**になった。

```
NDA締結後の変化：
・企業っぽいアカウントからの閲覧増加
・「NDAありますか？」という問い合わせ1件
```

**初めての問い合わせ！**

結局その案件は流れたけど、手応えを感じた。

### スキル・経歴を全部書く

```
Before（30文字）：
Unity開発できます。よろしくお願いします。

After（800文字）：
【経歴】
・2020年〜：Unity独学開始

・Unity 2021/2022 LTS
・Visual Studio

【対応可能な案件】
・ハイパーカジュアルゲーム
・VRコンテンツ
・AR機能実装
・既存プロジェクトの改修
```

**数字と具体性**が大事だと学んだ。

## 競合分析で見つけた「穴場」

毎日1時間、**売れてるサービスを分析**した。

エクセルに記録：
```
| サービス名 | 価格 | 販売数 | 評価 | 特徴 |
|-----------|------|--------|------|------|
| A社 | 3,000円 | 156件 | 4.9 | 納期明記 |
| B社 | 10,000円 | 89件 | 5.0 | VR特化 |
| C社 | 1,000円 | 234件 | 4.7 | 激安 |
```

### 気づいた3つの共通点

**❶ タイトルに数字が入ってる**
- 「3日で納品」
- 「修正3回まで無料」
- 「100件以上の実績」

**❷ 納期を明確にしてる**
- みんな「要相談」が多い中、明記してる人が売れてた

**❸ 専門分野を絞ってる**
- 「なんでもやります」より「VR専門」の方が売れてる

### 見つけた穴場：超短納期

分析してて気づいた。
**「今すぐ欲しい」人が結構いる。**

でも、ほとんどのサービスは納期1週間〜。

> 「じゃあ、3日納期で勝負しよう」

タイトル変更：
```
Before：Unity開発承ります
After：【最速3日】Unity開発｜学生ですが全力対応
```

**閲覧数が2倍になった。**

## プライドを捨てて500円にした瞬間

でも、まだ問い合わせは来ない。

5月10日。
ココナラ登録から1ヶ月以上経過。

> 「もう、プライドとか言ってる場合じゃない」

**価格：5,000円 → 500円**

10分の1。
正気じゃないと思った。

でも翌日、**初めてDMが来た。**

## 時給12.5円でも嬉しかった初案件

```
「Unityでシューティングゲーム作れますか？
予算500円ですが...」
```

即レスした。

> 「はい！できます！ぜひやらせてください！」

内容：
- 2Dシューティングゲーム
- 敵3種類
- ボス1体
- スコアシステム

**普通なら3万円の案件。**

でも受けた。
最初の実績が欲しかったから。

### 40時間かけて作った

朝から晩まで、3日間缶詰め。

```
1日目：12時間（基本システム）
2日目：15時間（敵AI・ボス）
3日目：13時間（調整・バグ修正）
合計：40時間
```

**時給換算：12.5円**

コンビニバイトの**1/80**。

### でも評価5.0がもらえた

納品したら、めちゃくちゃ喜んでもらえた。

```
「想像以上のクオリティでした！
また機会があればお願いします！」
評価：★★★★★
```

**この瞬間、全てが変わった。**

評価が1つ付いただけで、閲覧数が跳ねた。
```
評価0 → 1になった翌週：
閲覧数：15 → 67
お気に入り：0 → 3
```

## 2件目は3,000円で受注できた

評価が付いたら、すぐ値上げした。

**500円 → 3,000円**

「また誰も買わなくなるかも...」

不安だったけど、1週間後に問い合わせが来た。

```
「パズルゲーム作れますか？
予算3,000円です」
```

**6倍の単価で受注！**

しかも作業時間は10時間で済んだ。
（シューティングの素材が使い回せたから）

**時給300円。まだ安いけど、成長を感じた。**

## 振り返って分かった「最初の1件」の重要性

今、230件以上受注して分かる。

**最初の1件が一番大変。**

でも、最初の1件さえ取れれば：
- 評価が信頼になる
- 次の案件が取りやすくなる
- 単価を上げても売れる

### もし今、最初の1件で悩んでるなら

**赤字覚悟でいい。**

500円でも、100円でも。
最初の評価を取ることに全力を注げ。

プライドは後から取り戻せる。
でも**チャンスは待ってくれない**。
