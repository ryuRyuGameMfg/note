2022年4月、**時給12.5円**からスタート。
2024年12月、**月収70万円**を達成。

単価を**100倍**にした方法を、包み隠さず書く。

多くの人が「安くしないと売れない」と思ってる。
**それ、間違い。**

価格は、自分で決められる。
戦略次第で、いくらでも上げられる。

## 価格を上げる勇気が出なかった

500円で初受注した後、すぐ3,000円に値上げした。

でも、そこから先が怖かった。

> 「5,000円にしたら誰も買わないかも...」

ビビって、3ヶ月間も3,000円のまま。

当時の収支：
```
単価：3,000円
月間受注：15件
売上：45,000円
作業時間：150時間
時給：300円
```

**コンビニバイトの方がマシ。**

## 評価が増えるたびに値上げする法則

転機は、ある先輩フリーランスの言葉。

> 「評価10件ごとに、値段倍にしろ」

半信半疑だったけど、やってみた。

```
評価0件：500円
評価10件：3,000円（6倍）
評価30件：8,000円（2.6倍）
評価50件：15,000円（1.9倍）
評価100件：30,000円（2倍）
評価150件：50,000円（1.7倍）
```

**誰も文句言わなかった。**

むしろ、高い方が信頼される。

### 値上げのタイミングと理由

```
タイミング：評価が10の倍数になった時
理由：「好評につき」「実績増加のため」

告知方法：
・サービスページに記載
・1ヶ月前から予告
・既存客には旧価格適用
```

**段階的な値上げが、受け入れられやすい。**

## 松竹梅の法則で平均単価を上げる

単一価格じゃなく、**3つの選択肢**を用意。

### 2023年1月時点の価格設定

```
🥉 ライトプラン：15,000円
   ・簡単な機能追加
   ・修正3回まで
   ・納期1週間

🥈 スタンダード：50,000円（70%が選ぶ）
   ・ミニゲーム開発
   ・修正無制限
   ・納期1週間
   ・1ヶ月サポート

🥇 プレミアム：150,000円
   ・フルゲーム開発
   ・要件定義から対応
   ・納期調整可能
   ・3ヶ月サポート
   ・優先対応
```

**真ん中が一番売れる。**

これ、行動経済学で証明されてる。

### 松竹梅の心理効果

```
ライトだけ：平均単価15,000円
松竹梅あり：平均単価48,000円
```

選択肢を増やしただけで、**単価3倍**。

## 時間単価を意識した価格設定

大事なのは、**時給換算**。

```
【2022年4月】初期
案件：500円（40時間）
時給：12.5円

【2023年1月】中期
案件：50,000円（10時間）
時給：5,000円

【2024年12月】現在
案件：100,000円（8時間）
時給：12,500円
```

**時給が1,000倍。**

### 効率化で時給を上げる

同じ50,000円でも：
```
効率化前：20時間 → 時給2,500円
効率化後：5時間 → 時給10,000円
```

**値上げより、効率化の方が楽。**

## 安い仕事を断る勇気

2023年6月、決断した。

**「3万円以下の仕事は受けない」**

```
断った案件：
「予算5,000円でRPG作れますか？」
「1万円で3Dアクションゲーム」
「修正だけ2,000円でお願い」
```

### 断り方のテンプレート

```
お問い合わせありがとうございます。

大変申し訳ございませんが、
現在のスケジュールと価格設定では
お受けすることが難しい状況です。

もしご予算の調整が可能でしたら、
以下の条件でお受けできます：

・最低料金：30,000円〜
・納期：2週間〜

ご検討いただければ幸いです。

他の制作者様もご検討されることを
おすすめいたします。
ココナラには素晴らしい制作者が
たくさんいらっしゃいます。

ご縁がありましたら、
またお声がけください。
```

**丁寧に、でもきっぱりと。**

### 断ることで起きた変化

```
断る前：
・月30件受注
・平均単価20,000円
・売上600,000円
・労働時間300時間

断った後：
・月15件受注
・平均単価50,000円
・売上750,000円
・労働時間120時間
```

**半分の仕事で、売上25%増。**

## 高単価を正当化する方法

高い値段つけるなら、**理由**が必要。

### ❶ 専門性を前面に出す

```
Before：「Unity開発します」
After：「VR/AR専門のUnity開発」
```

**専門家は高くて当然。**

### ❷ 付加価値をつける

```
基本サービスに含まれるもの：
✅ ソースコード完全提供
✅ 詳細なコメント付き
✅ 1ヶ月無料サポート
✅ 今後の拡張提案書
✅ 著作権譲渡
```

**「高いけど、これなら納得」を作る。**

### ❸ 実績で説得する

```
サービスページに記載：
・累計230件以上の開発実績
・大手企業案件も対応
・リピート率60%
・平均評価4.9
```

**数字が、価格を正当化する。**

## 価格戦略で学んだ5つの真実

### 1. 安さは正義じゃない
安いと、逆に怪しまれる。

### 2. 高い方が楽な客が来る
安い客ほど要求が多い。

### 3. 断ることで価値が上がる
「誰でもOK」は価値が低い。

### 4. 時給で考える癖をつける
売上より時給が大事。

### 5. 値上げは定期的に
インフレもある。遠慮するな。
