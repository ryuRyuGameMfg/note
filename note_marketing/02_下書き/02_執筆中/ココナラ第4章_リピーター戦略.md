**リピーターを増やせばいい。**

## 納品して終わり、が一番もったいない

最初の頃の僕：

```
納品完了しました。
ありがとうございました。
```

**これで終わり。**

せっかく満足してもらったのに、
次の依頼に繋げる努力ゼロ。

これじゃリピーター増えない。

## 60分以内返信が信頼を作る

ココナラで一番大事なのは**レスポンス速度**。

データ取ってみた：

```
平均返信時間とリピート率：
〜30分：リピート率72%
〜1時間：リピート率65%
〜3時間：リピート率45%
〜6時間：リピート率30%
翌日以降：リピート率10%
```

**30分以内に返信すると、リピート率7倍。**

### 通知設定を極める

```
スマホ設定：
・ココナラアプリ通知：ON
・メール通知：ON
・Apple Watch連携：ON
・通知音：最大

PC設定：
・ブラウザ通知：ON
・デスクトップ通知：ON
・Gmail転送設定：ON
```

うざいくらい通知来る。
でも**これが稼ぎに直結**。

### 定型文で速度を上げる

よく使う返信は、辞書登録。

```
「おつ」→「お世話になっております。RYURYUです。」
「かく」→「確認いたしました。」
「ごれ」→「ご連絡ありがとうございます。」
「しょう」→「承知いたしました。すぐに対応いたします。」
```

これで返信速度**3倍**。

**送った人の30%が、追加依頼をくれる。**
