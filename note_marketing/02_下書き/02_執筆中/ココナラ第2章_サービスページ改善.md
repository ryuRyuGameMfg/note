でも評価が付いたことで、**流れが変わり始めた**。

ただ、まだ問題があった。

> 「閲覧数は増えたけど、クリックされない...」

100人見ても、1人しかサービスページを開かない。
**クリック率1%**。

これじゃ稼げない。

その方法を全部公開する。

## サムネイルを変えただけで閲覧数2倍

最初のサムネイル、恥ずかしすぎる。

**Unityのロゴだけ。**

白背景に、Unityの公式ロゴ置いただけ。
素人丸出し。

> 「これじゃ誰もクリックしないわ...」

### Canvaで30分で作れるプロ級サムネイル

無料ツール**Canva**を使い始めた。

テンプレート検索：「ビジネス」「サービス」
気に入ったやつを選んで、カスタマイズ。

```
改善前：Unityロゴだけ
改善後：
・背景：青→紫のグラデーション
・メインコピー：「最速3日納品」（特大サイズ）
・サブコピー：「Unity開発」
・実績バッジ：「50件突破」（右上に配置）
・アイコン：VR/ARのピクトグラム
```

**ポイント：文字は大きく、情報は少なく。**

スマホで見ると、サムネイルちっちゃい。
細かい文字なんて読めない。

### 色の心理学を使った

```
青：信頼感・安心感
紫：クリエイティブ・高級感
オレンジ：行動喚起（CTAボタン的な）
```

僕は**青×紫のグラデーション**にした。

競合はみんな似たような色使い。
**差別化するなら色から。**

### A/Bテストの結果

1週間ごとにサムネイル変えて、データ取った。

```
週1：Unityロゴのみ → クリック率0.8%
週2：青背景+文字 → クリック率1.2%
週3：グラデ+実績 → クリック率1.8%
週4：最終版 → クリック率2.3%
```

**サムネイルだけで3倍の差。**

## タイトル29文字の法則

ココナラのタイトル、**最大40文字**。

でも実は、**検索結果では29文字**までしか表示されない。
（スマホだと25文字）

### ダメなタイトル例

最初につけてたタイトル：
```
「Unity開発承ります！2D/3D対応可能です！よろしくお願いします」
```

38文字。
しかも、**インパクトゼロ**。

### 売れるタイトルの方程式

```
【フック】＋メインキーワード＋｜サブキーワード
```

実例：
```
【最速3日】Unity2D/3Dゲーム開発｜VR/AR対応
```

29文字ジャスト。

**【】で目立たせる**のがポイント。
検索結果でめちゃくちゃ目立つ。

### キーワードの選び方

Googleキーワードプランナーで調査した。

```
検索ボリューム（月間）：
・Unity開発：1,200
・Unity ゲーム：800
・Unity VR：600
・Unity 依頼：400
・Unity 外注：350
```

**「Unity開発」は必須。**

でも競合も多い。
だから**「VR/AR」で差別化**した。

## サービス説明で購入率が変わる

タイトルでクリックさせても、
サービス説明がダメなら買われない。

### 冒頭3行で決まる

ココナラのサービス説明、**冒頭3行**しか表示されない。
「もっと見る」を押さないと、全文読めない。

```
ダメな例：
はじめまして。Unity開発を承っております。
よろしくお願いいたします。
お気軽にご相談ください。
```

**情報ゼロ。離脱確定。**

```
良い例：
✅ 最短3日で納品可能
✅ 修正無制限（納品後1週間）
✅ VR/AR開発の実績50件以上
```

**メリットを箇条書き。これが鉄則。**

### 絵文字は適度に使う

絵文字使いすぎると**素人っぽい**。
使わなすぎると**堅い**。

バランスが大事。
見出しに1個、リストマークとして使うくらいがちょうどいい。

## PR動画で差をつける

ココナラは**動画**も設定できる。
これ、やってる人少ない。

**だからこそチャンス。**

### スマホで撮影した30秒動画

高い機材なんていらない。
iPhoneの画面録画で十分。

```
動画構成（30秒）：
0-5秒：自己紹介（顔出しなし、声だけ）
「Unity開発のRYURYUです」

5-15秒：実績作品を画面録画で紹介
「これまで50件以上の開発実績があります」

15-25秒：対応可能な技術を文字で表示
・2D/3Dゲーム
・VR/AR
・マルチプラットフォーム

25-30秒：CTA
「お気軽にご相談ください」
```

編集は**無料アプリ**（CapCut）でサクッと。

### 動画の効果はすごかった

```
動画なし：問い合わせ率2%
動画あり：問い合わせ率5%
```

**2.5倍の差。**

動画があるだけで、**信頼感が全然違う**。
