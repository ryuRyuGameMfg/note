## 【第4章】ChatGPTが苦手な5つのエラーパターン

### ❶ Unity固有のワークフロー問題

**Prefabのオーバーライド**とか**シリアライゼーション**とか、
Unity特有の概念はChatGPTの理解が浅い。

先週も「Prefab内のスクリプト参照が外れる」問題で、
ChatGPTは一般的なC#の話しかしてくれなかった。

### ❷ バージョン依存の問題

```
Unity 2021 → 2023にアップグレード後、
Input Systemが動かない
```

ChatGPTは最新バージョンの仕様変更を**リアルタイムで把握してない**。

### ❸ パッケージの組み合わせ問題

- URP + Post Processing
- XR Toolkit + New Input System
- Cinemachine + Timeline

**パッケージ同士の相性問題**は、公式ドキュメントにも載ってない。

### ❹ プラットフォーム固有エラー

iOSビルドで**"Symbol not found"**とか、
Androidで**"Unable to merge manifests"**とか。

これらはChatGPTじゃなく、**Stack OverflowかUnity Forum**の方が早い。

### ❺ 最新機能・実験的機能

- Unity Muse
- Sentis
- Netcode for Entities

**2024年以降の機能**は、ChatGPTの学習データに含まれてない。
