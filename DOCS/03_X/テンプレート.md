# Xテンプレート

参照: 04_SEO/公式アルゴリズム情報.md

## アルゴリズム対応ポイント

1. 返信 = いいねの27倍の価値 → 質問で締める
2. 外部リンクは低評価 → 返信欄に配置
3. 画像付きは1.7倍拡散 → 画像を添付

## 日常・開発進捗型（返信獲得狙い）

```
今日の開発メモ。

〇〇の機能、ようやく動いた！
△△時間かかったけど、□□で解決。

皆さんは〇〇で困った経験ありますか？

#Unity #ゲーム開発
```

ポイント:
- 最後に質問で締める（返信=27倍の価値）
- URLは入れない（入れる場合は返信欄）
- 画像があればなお良い

## 失敗談・学び型（共感獲得狙い）

```
正直に言う。

最初の見積もり、甘すぎた。
時給換算したらバイト以下だった...

今は〇〇を意識してる。

同じ経験ある人いますか？

#個人開発 #フリーランス
```

ポイント:
- ネガティブ→学びの構成
- 共感を呼ぶ→返信が来やすい
- 質問で締める

## Tips・気づき型（保存獲得狙い）

```
個人開発で大事だと思うこと。

・〇〇する
・△△を意識
・□□を避ける

これだけで□□件やってこれた。

保存しておくと便利かも

#個人開発 #Unity
```

ポイント:
- 箇条書きで見やすく
- 「保存」を促す
- 画像（図解）があると保存率UP

## やりましたよ報告型

```
やったー！〇〇達成！

△△ヶ月かかったけど、
ようやくここまで来れた。

次は□□を目指します。

応援してくれた皆さんありがとう！

#達成報告 #個人開発
```

ポイント:
- 感情を込める
- 数字で具体的に
- 感謝を伝える

## note記事告知型（リンクは返信欄）

本文:
```
【新記事】〇〇について書きました！

・△△の方法
・□□のコツ
・実際の結果（数字あり）

詳細はリプ欄に

#note #ゲーム開発
```

返信欄に投稿:
```
記事はこちら：
https://note.com/xxx/n/xxx
```

ポイント:
- 本文にURLを入れない（アルゴリズム低評価）
- 「リプ欄に」と明記
- 記事の要点を3つ程度

## サービス告知型（リンクは返信欄）

本文:
```
【お知らせ】〇〇サービス始めました！

できること：
・△△
・□□
・◇◇

235件の実績を活かして対応します。

詳細はリプ欄かプロフから

#ココナラ #Unity
```

返信欄に投稿:
```
サービスページ：
https://coconala.com/services/xxx
```

ポイント:
- 本文にURLを入れない
- 宣伝と日常の比率2:8を守る
- 実績の数字を入れる

## 質問募集型（エンゲージメント狙い）

```
質問募集！

Unity/ゲーム開発について
聞きたいことあればリプで教えてください。

できる範囲で答えます！

#Unity #ゲーム開発 #質問募集
```

ポイント:
- 返信を直接促す
- 専門性をアピール
- フォロワーとの関係構築

```

〇〇については△△がポイントです。

他にも気になることあれば聞いてください！
```

```

〇〇、共感してもらえて嬉しいです。
また△△についても書く予定なので、

```

```

```

## テンプレート使用時の注意

| 項目 | 対応 |
|------|-----|
| 質問で締める | 返信=いいねの27倍 |
| 外部リンク | 本文NG、返信欄へ |
| 画像・動画 | 1.7倍拡散効果 |
| 返信対応 | 必ず返す（返信への返信=150倍） |
