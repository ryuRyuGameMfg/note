# リリースツール（旧版・非推奨）

⚠️ **このツールは非推奨です。`thumbnail-generator` に統合されました。**

## 新しいツールを使用してください

**`thumbnail-generator`** が統合版として機能しています。

```bash
cd thumbnail-generator
npm run dev
# http://localhost:3005 で起動
```

詳細は `thumbnail-generator/README.md` を参照してください。

## 旧版の情報（参考）

このフォルダは旧版のリリースツールです。以下の機能がありました：

- 記事一覧表示（ポート8080）
- サムネイルジェネレーターへのリダイレクト（ポート3000）

現在は `thumbnail-generator` に統合され、単一のアプリケーションで全ての機能が利用できます。
