# SEO/AEO チェックリスト（note/Google/AI検索対応）

## note向け（公開設定/記事構造）
- [ ] タイトルにメインKW＋具体数字（期間/金額/件数）
- [ ] 冒頭200字にメインKW自然挿入
- [ ] 見出し（##/###）にサブKWを散りばめる
- [ ] タグは5個以内で関連性高いKW
- [ ] 内部リンク（関連note）を2–3本
- [ ] 画像は適切なキャプション（alt代替）

## Google向け（2025時点の原則）
- [ ] E-E-A-Tを提示（経験/専門性/権威/信頼）
- [ ] YMYL該当時は一次情報/出典/免責を明記
- [ ] 体験ベースの具体性（数値・期間・条件）
- [ ] 重複・薄い内容を避け、テーマの意図を網羅
- [ ] モバイル可読・改行3–4行・CTA邪魔しない

## AEO向け（AI/音声・生成検索最適化）
- [ ] Q&A/FAQを2–5問用意（質問→1〜3文の直接回答）
- [ ] 定義・手順・比較は箇条書き短文で要約可能に
- [ ] エンティティ（固有名詞）を正式名で明記
- [ ] スキーマ採用可能時はFAQ/HowTo/Articleを検討
- [ ] 結論先出しの要約（冒頭）を100–200字で用意

## コピペ安定/ノート仕様
- [ ] 引用ルール（note_markdown_安定運用ルール.md）を満たす
- [ ] URLは独立行、前に説明文
- [ ] 水平線（---）は避ける
