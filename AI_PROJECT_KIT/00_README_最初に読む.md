# AI_PROJECT_KIT（コピペ運用キット）

目的：このフォルダを**任意の開発プロジェクトのルートにコピペ**するだけで、AI（Cursor/Copilot等）を使った開発運用をすぐ開始できるようにする。

## 使い方

1. この `AI_PROJECT_KIT/` を、対象プロジェクトのルートにコピペ
2. まず `10_AI指示書_プロジェクト雛形.md` の **「埋める欄」** を埋める
3. 次に `30_タスク依頼テンプレ.md` を使ってタスク投入
4. PR作業は `40_PRテンプレ.md` と `50_チェックリスト.md` を使って回す

## このキットの設計思想

- **AIは高速実装者（要監督）**：仕様は人間が握る
- **小さいPR**：1PR=1目的
- **検証優先**：テスト/型/lint を通す
- **ガードレール**：秘密情報/破壊的操作/外部送信を制限
- **計画承認後に実装**：AIが暴走しない
